#  Vansh Portfolio Website

A modern personal portfolio website built using Flask, HTML, CSS, JavaScript, and OpenRouter AI. This website showcases my projects, technical skills, achievements, and includes an AI-powered chatbot that can answer questions about me and my work.

## Features

* Modern Responsive UI
* Flask Backend
* OpenRouter AI Chatbot
* About Me Section
* Skills Showcase
* Services Section
* Projects Showcase
* Resume Download
* Contact Information
* Mobile-Friendly Design

## 🛠️ Tech Stack

* JavaScript

### Backend

* Flask
* Python

### AI Integration

* OpenRouter API
* Large Language Models (LLMs)

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/vansh1236/Vansh-Portfolio.git
```

Move into the project folder:

```bash
cd Vansh-Portfolio
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
